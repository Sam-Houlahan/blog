Post by Sam Houlahan, February 27, 2017.

What is a grid based design?

A grid based design is a method used in design and web development. A grid design is used to improve consistency over the website.
This system is generally used when creating a responsive website. Using a css framework such as Bootstrap will make this alot easier and will give you a professional responsive website.
Most websites use a grid based design,well all the websites I investigated had a grid based design. Generally speaking you would use a 12 column grid in order to have a lot of flexibility with your website.

What is all the hype behind responsive webpages?

We are in a day and age where people generally have multiple devices with different screen sizes. Meaning that websites now need to be catered to all the devices otherwise your website will end up with less traffic. It is also the most cost effective way to build your website rather than spending resources to make a seperate mobile website you can just create one responsive site.

What is semantic structure?

Using a semantic structure is important for web development as it will help the computer make meaning of your content. HTML5 includes tags called semantic elements, they give information about the type of content in these elements. For example if you have a footer you would use the semantic tag "footer".
