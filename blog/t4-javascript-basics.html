Javascript Basics

Posted by Sam Houlahan, March 16 2017.

How does JavaScript compare to HTML and CSS?

HTML and CSS let you build and style the website, where as Javascript is the language used for user interactions and implementing features. The main difference I have found is Javascript allows for complex logic such as control flow and loops!.

Explain control flow and loops using an example process from everyday life, for example 'waking up' or 'brushing your teeth'.

In programming a control flow is referred too as a set of statement's that are executed depending on what conditions are met. An example of this in every day life would be a situation like going out to the movies. If you were in the mood for comedy you would choose a movie from that category, If you wanted to be scared you would choose from the horror category. Else if you were not interested in going out to a movie you would stay home.

A loop is a process of repeating the same action until the condition is met. An example of this in everyday life would be the action of washing dishes. You repeat a washing/scrubbing action untll the condition that the dish is completely clean is met. In theory if this loop was in a function you could also add the number of dishes as a parameter and set it up so that the loop knows how many dishes need to be cleaned before it finishes.

Explain the difference between accessing data from arrays and objects.

With an array to access the data you would keep in mind that array's data indexing starts from 0 from here you specify the number of the element you want then surround it with brackets.
Example:
var myArr = ["Sam",23];
Then to access the 2nd element you would use myArr[1];

To access data from an object you can use 2 different ways.
The first is the similar as accessing data from an array the second is using a dot notation.
Example:
var myObj ={ firstname : "Sam" secondname : "Houlahan"};
First way to access myObj[firstname];
Second way to access myObj.firstname;
