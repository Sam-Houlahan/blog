Javascript Language

Posted by Sam Houlahan, March 30 2017.

In the JavaScript language, describe all the use cases for each of:

Parentheses ()

  • Used to execute a function doSomething()
  • Used for containing expressions if (a > 10)
  • Used when parameters are required sayHi(greeting,name)

Brackets []

  • Can be used to access an Object's properties myObj['1']
  • Used for making an array myArr = [ 1,2,3]
  • Used for accessing elements from an array or string myArr[0] would return 1

Braces {}

  • Used to create objects in your program var myName = { first:'Sam', last:'Houlahan' }
  • Used to enclose blocks of code such as functions,conditional blocks,loops
    for(i=0; i< 10 ; i++){ console.log(i)}

Single quotes ''

  • Used when declaring a string 'hello'

Double quotes ""

  • Can also be used when declaring a string "hello"
  • JSON notations require double quotes
