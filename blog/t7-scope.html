What is 'scope', and how does it work in Javascript?

No it is not meaning a sniper scope!.Scope in programming in general is referring to where you can use your variables in your code. In terms of Javascript it is referring to the global scope and the local scope.

Global Scope:
All variables declared outside of a function are defaulted to the global scope which means you can use them wherever you may please in your program.

let imGlobal = 'hi';
function checkIfGlobal(){
console.log(imGlobal); }
checkIfGlobal(); // hi
